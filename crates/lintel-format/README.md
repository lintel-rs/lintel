Format command for [Lintel](https://github.com/lintel-rs/lintel). Provides the `lintel format` subcommand which formats JSON, JSONC, JSON5, YAML, Markdown, and TOML files using prettier-compatible rules.

Supports `--check` mode for CI and `--exclude` patterns for skipping files.
