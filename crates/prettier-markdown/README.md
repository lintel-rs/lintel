Prettier-compatible Markdown formatter

Prettier-compatible Markdown formatter.

Formats Markdown content including YAML frontmatter extraction and formatting. Uses the Wadler-Lindig pretty-printing algorithm with support for prose wrap, print width, and other prettier options.
