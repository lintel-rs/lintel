Prettier configuration resolution and types

Prettier-compatible formatting configuration types and config file resolution.

Provides the `PrettierConfig` struct with all prettier 3.x options (print width, tab width, trailing commas, quote style, prose wrap, etc.) and utilities to resolve `.prettierrc` files.
