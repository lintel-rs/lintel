Prettier-compatible code formatter dispatcher

Prettier-compatible formatter for JSON, JSONC, JSON5, YAML, and Markdown.

Unified formatting API that dispatches to format-specific crates. Includes config resolution from `.prettierrc` files and format detection from file extensions.
