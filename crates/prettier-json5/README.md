Prettier-compatible JSON5 formatter

Prettier-compatible JSON5 formatter.

Formats JSON5 content preserving comments, with support for trailing commas, quote style, bracket spacing, and other prettier options. Uses the Wadler-Lindig pretty-printing algorithm.
