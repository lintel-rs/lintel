Prettier-compatible JSON/JSONC formatter

Prettier-compatible JSON and JSONC formatter.

Formats JSON and JSONC (JSON with comments) content with support for trailing commas, bracket spacing, and other prettier options. Handles both strict JSON and JSONC via format detection.
