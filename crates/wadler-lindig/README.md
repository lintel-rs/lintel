Wadler-Lindig pretty-printing algorithm

Implementation of the Wadler-Lindig pretty-printing algorithm.

Provides a document IR (`Doc`) with combinators for text, groups, indentation, alignment, line breaks, and fill layout. The printer resolves flat-vs-broken decisions against a configurable print width.
